# 功能引擎：庫存與購物車管理 (Skills)

## 1. 基礎實作規劃

本技能模組負責確保顧客點餐的商品與門市實際備料能力一致，處理購物車生命週期：

### 核心運算步驟

- **新增至購物車**：當顧客點擊加入購物車時，系統自動將數量轉換為對應配料扣除量（例如 2 杯珍珠即扣除 2 份珍珠庫存）。若庫存低於 0，則中斷操作。
- **購物車修改/刪除**：若顧客於購物車內取消某項飲品，系統須讀取該飲品的配料量，並如數歸還至門市庫存池。
- **迴圈點餐維持**：管理點餐流程狀態 (State Array)，使顧客在同一筆訂單下可不斷新增商品。

## 2. 進階實作規格 (AI好理解版)

| 輸入操作 | 系統行為 | 狀態變更 (State Mutation) |
| :--- | :--- | :--- |
| **Add_Item** | 確認庫存 `->` 新增品項 | `Cart.push(item)`, `Inventory.boba -= qty` |
| **Remove_Item** | 尋找 ID `->` 移除品項 | `Cart.splice(index)`, `Inventory.boba += qty` |
